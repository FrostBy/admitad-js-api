# Параметры limit, offset

Параметры пагинации для управления количеством и позицией возвращаемых записей.

## Параметры

| Параметр | По умолчанию | Максимум | Описание |
|----------|:------------:|:--------:|----------|
| `limit` | 20 | 500 | Количество записей в ответе |
| `offset` | 0 | — | Смещение от начала списка |

## Использование

```
GET /api/method/?limit=50&offset=100
```

## Пагинация

| Страница | Параметры |
|:--------:|-----------|
| 1 | `offset=0&limit=20` |
| 2 | `offset=20&limit=20` |
| 3 | `offset=40&limit=20` |
| N | `offset=(N-1)*limit&limit=20` |

## Формат ответа

```json
{
  "results": ["..."],
  "_meta": {
    "count": 150,
    "limit": 20,
    "offset": 0
  }
}
```

| Поле | Описание |
|------|----------|
| `results` | Массив записей |
| `_meta.count` | Общее количество записей |
| `_meta.limit` | Текущий лимит |
| `_meta.offset` | Текущее смещение |
