# Параметр language

Параметр для выбора языка ответов API.

## Использование

```
GET /api/method/?language=en
```

## Поддерживаемые языки

| Код | Язык |
|:---:|------|
| `ru` | Русский (по умолчанию) |
| `en` | Английский |
| `es` | Испанский |
| `tr` | Турецкий |
| `pl` | Польский |

## Примечание

Если параметр не указан, используется русский язык (`ru`).
