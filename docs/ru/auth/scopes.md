# Справка по настройкам доступа (Scopes)

Права доступа определяют, какие методы API доступны приложению.

## Публичные данные

| Scope | Описание |
|-------|----------|
| `public_data` | Типы площадок, языки, категории, регионы, валюты, категории купонов |

## Площадки

| Scope | Описание |
|-------|----------|
| `websites` | Чтение списка площадок пользователя |
| `manage_websites` | Создание, редактирование, подтверждение и удаление площадок |

## Партнёрские программы

| Scope | Описание |
|-------|----------|
| `advcampaigns` | Доступ к списку партнёрских программ |
| `advcampaigns_for_website` | Просмотр программ для конкретной площадки |
| `manage_advcampaigns` | Подключение и отключение партнёрских программ |

## Баннеры и лендинги

| Scope | Описание |
|-------|----------|
| `banners` | Просмотр баннеров |
| `banners_for_website` | Баннеры для конкретной площадки |
| `landings` | Доступ к лендингам |

## Купоны

| Scope | Описание |
|-------|----------|
| `coupons` | Список всех купонов |
| `coupons_for_website` | Купоны для конкретной площадки |

## Профиль и баланс

| Scope | Описание |
|-------|----------|
| `private_data` | Данные профиля (имя, язык) |
| `private_data_email` | Профиль + электронная почта |
| `private_data_phone` | Профиль + телефон |
| `private_data_balance` | Информация о балансе |

## Ссылки

| Scope | Описание |
|-------|----------|
| `validate_links` | Тестирование ссылок |
| `deeplink_generator` | Генератор deeplink |
| `short_link` | Сокращение ссылок |
| `broken_links` | Список битых ссылок |
| `manage_broken_links` | Решение проблем с битыми ссылками |

## Статистика

| Scope | Описание |
|-------|----------|
| `statistics` | Статистика и отчёты |

## Оптимизация (Postback)

| Scope | Описание |
|-------|----------|
| `opt_codes` | Просмотр кодов оптимизации |
| `manage_opt_codes` | Создание и редактирование postback-ов |

## Retag

| Scope | Описание |
|-------|----------|
| `webmaster_retag` | Просмотр тегов retag |
| `manage_webmaster_retag` | Управление тегами retag |

## Заказы

| Scope | Описание |
|-------|----------|
| `lost_orders` | Просмотр потерянных заказов |
| `manage_lost_orders` | Создание потерянных заказов |

## Брокер

| Scope | Описание |
|-------|----------|
| `broker_application` | Список заявок брокера |
| `manage_broker_application` | Создание заявок брокера |

## Прочее

| Scope | Описание |
|-------|----------|
| `announcements` | Список оповещений |
| `referrals` | Информация о рефералах |
| `aliexpress_commission` | Расчёт вознаграждения за товары AliExpress |
| `vendor_tool` | Активные и запланированные вендор-бонусы |
| `web_notificator` | Получение списка уведомлений |
